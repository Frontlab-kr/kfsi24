시험(심사) 신청내역

나의 시험신청 내역

  • 1급 / 2급 / 3급 소방안전관리자 시험의 일정변경 또는 응시수수료 환불기준
    - 해당 시험시행일 전일까지 신청한 경우 일정변경 또는 전액환불 (시험시행일부터 환불 불가)
  • 특급 소방안전관리자 시험의 일정변경 또는 응시수수료 환불기준
    - 해당 시험시행일 5일 전까지 신청한 경우 일정변경 또는 전액환불 (시험시행일 4일 전부터 환불 불가)
  • 단, 일정변경은 해당연도 시험일정이 있는 경우에 한하며, 직접 방문하거나 인터넷 등으로 신청하여야 합니다.
    방문신청의 경우 근무일 중에만 가능합니다.
  • "지난 신청 내역" 중 해당 시험일로부터 3개월까지의 신청내역만 조회됩니다.
나의 강의실 바로가기
사용자가 신청한 시험의 내역을 확인하는 테이블입니다. 응시번호, 시험과정, 지부, 시험일자, 상태, 약도 정보를 제공합니다.

온라인 응시자격 심사신청 내역

  • 1급 / 2급 / 3급 소방안전관리자 시험의 일정변경 또는 응시수수료 환불기준
    - 해당 시험시행일 전일까지 신청한 경우 일정변경 또는 전액환불 (시험시행일부터 환불 불가)
  • 특급 소방안전관리자 시험의 일정변경 또는 응시수수료 환불기준
    - 해당 시험시행일 5일 전까지 신청한 경우 일정변경 또는 전액환불 (시험시행일 4일 전부터 환불 불가)
  • 단, 일정변경은 해당연도 시험일정이 있는 경우에 한하며, 직접 방문하거나 인터넷 등으로 신청하여야 합니다.
    방문신청의 경우 근무일 중에만 가능합니다.
  • "지난 신청 내역" 중 해당 시험일로부터 3개월까지의 신청내역만 조회됩니다.
  • 온라인 응시자격 심사 승인은 신청일로부터 5~7일 정도 소요됩니다.
  • 응시자격 심사 승인 이후 ‘시험신청‘ 메뉴에서 시험접수 가능합니다.
  • 안전원 담당자가 접수한 이후에는 심사신청 내역 수정 및 취소가 불가능합니다.
  • 안전원 심사 과정에서 사진, 증명서류 등 불분명한 경우 보완을 요청할 수 있습니다.
  • 보완요청 사항에 대해서는 3일 이내에 수정하여 주시기 바랍니다. (기한 내 미수정 시 자동취소 될 수 있습니다.)
