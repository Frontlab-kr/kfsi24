원격교육 안내 동영상 보기
  • 원활한 접수를 위해 '크롬' 브라우저를 이용 바랍니다.
  • 시험관련 메뉴를 이용하는 경우에는 '실명확인 로그인'을 이용해 주시기 바랍니다.
  • 시험일정의 잔여인원 및 접수를 하고자 할 때에는 '일정 새로고침'(또는 키보드의 'F5')을 클릭하여야 새로운 정보가 반영됩니다.
  • (예시) 10:00시 접수 시 10:00시에'일정 새로고침' 버튼 또는 키보드의 'F5' 버튼으로 새로고침을 진행해야 접수가 가능합니다.
  • 시험일정 변경(타 지부로의 일정변경 포함)을 하고자 할 때에는 '시험 신청 내역'에서 '일정변경 신청' 버튼을 눌러 일정확인 및 변경 가능합니다.

로그인 안내

  1. ZOOM 실행 [시작 > ZOOM >zoom]

  2. “ZOOM 클라우드 창”에서 “회의참가” 클릭
  3. “회의참가” 창에 회의 ID 및 이름에 정보입력
    • 상단에 “회의 참가” 창이 표시됨.
    • 첫 번째 입력란: 참가 ID [교육 접수 후 휴대폰 안내]
    • 두 번째 입력란: 수강번호(3자리)-성명
    • 옵션 체크박스: “이후 회의에서 내 이름 기억”, “오디오에 연결하지 않음”, “내 비디오 끄기” 옵션이 제공됨.
    • 하단에 파란색 “참가” 버튼과 회색 “취소” 버튼이 나열됨.
  4. “회의 암호 입력” 창에 회의암호 입력
    • “회의 암호 입력” 창이 화면에 표시됨.
    • 입력란에 회의 암호를 입력하는 필드가 있으며, 암호는 ***로 표시됨.
    • 하단에 파란색 “회의 참가” 버튼과 회색 “취소” 버튼이 나열됨.
    • 마우스 커서가 “회의 참가” 버튼 위에 위치함.
